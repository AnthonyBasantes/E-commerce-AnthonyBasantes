# ⚽🍔 Tony Food Stadium

Tony Food Stadium es un e-commerce de comida rápida con temática de fútbol, desarrollado como proyecto académico para la asignatura de Programación y Plataformas Web.

## 🚀 Características

- Diseño moderno inspirado en el fútbol.
- Catálogo de productos de comida rápida.
- Carrusel de imágenes.
- Información de la empresa.
- Integración con Google Maps.
- Enlace a LinkedIn del desarrollador.
- Diseño responsivo.

## 🛠 Tecnologías Utilizadas

- HTML5
- CSS3
- JSP
- Java
- Apache Tomcat
- GitHub
